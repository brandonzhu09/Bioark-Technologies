Plasmid Main Function Model Design Manual

The model design image represents the cassette model's functionality rather than its exact size and sequences. Customers only need to define the primary function of plasmids, such as the Cas9 plasmid or inducible expression cassette, as outlined above. Accessory cassettes, including gRNA or Tet-On protein expression, will be optimized and generated based on the selected main function plasmid.

Components within the main function vector will be automatically configured either as an integrated feature—such as a P2A-linked expression tag driven by the main promoter—or as a standalone cassette with its own promoter, based on internal optimization calculations. However, these features can be further customized after placing an order by accessing your customer account and submitting a modification request or by using the quote request file available at the bottom left link to specify your requirements at any time.

For the Standard and Lenti CRISPR groups, the functional kit consists of separate vectors, including Cas9 variant plasmids and gRNA plasmids. This design provides greater flexibility for applications and facilitates transfection. Notably, using separate vectors with a Lenti virus backbone can enhance viral titer and transfection efficiency.

For All-in-One, all functions are integrated into a single vector, ensuring a consistent ratio between Cas9 and sgRNA.

For Inducible Vectors, the expression of the Tet protein and the target expression cassette are separated in standard vectors but combined into a single vector in All-in-One versions. When using the separated function approach, you only need to design the primary function vector—the target expression vector—while the accessory Tet expression vector will be automatically matched.

For RNAi, mammalian, and protein expression, the standard vector is a standalone construct with full single-function capability.
